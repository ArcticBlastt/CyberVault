CyberVault is a cybersecurity-focused password manager built using Flask that enables users to securely store credentials, generate strong passwords, and manage their accounts with enhanced security. 
It implements user authentication with bcrypt hashing, session management, and a password vault for storing sensitive data. 
The application also includes an intrusion detection mechanism that tracks failed login attempts, blocks accounts after repeated failures within a defined time window, and provides activity logs for monitoring suspicious behavior. 
Designed with a clean cyberpunk UI, this project demonstrates practical concepts such as brute-force protection and real-time security monitoring in a web application environment.
